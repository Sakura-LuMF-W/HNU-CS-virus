# HNU-CS-virus
> A simple C-based virus spreading simulation project for Computer Systems course practice

## 📖 Introduction
This is an experimental project designed to review and reinforce the knowledge learned in the Computer Systems course. It simulates a simple, harmless virus spreading process to demonstrate core C programming concepts such as file I/O, process simulation, and data persistence.

## ✨ Features
- Simple, easy-to-understand implementation
- Modular design with clear separation of concerns
- No third-party dependencies, pure standard C implementation

## ⚙️ Environment
- **Build Tool**: GCC 9.4.0
- **Platform**: Linux (WSL Ubuntu 20.04+)
- **Dependencies**: Standard C library (stdio.h, stdlib.h, etc.)
- **Version Control**: Git 2.0+
